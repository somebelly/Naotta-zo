A world is a 3-tuple $(\Omega,A,\delta)$ such that:
  • $\Omega,A$ are two nonempty sets;
  • $\delta:\Omega\times A\to\{0,1\}$.
An element of $\Omega$ is called a state. An element of $A$ is called a thing. For $\omega\in\Omega$ and $a\in A$, we say thing $a$ exists in state $\omega$ if $\delta(\omega,a)=1$, and the opposite if $\delta(\omega,a)=0$.

In the rest of this daydream, $(\Omega,A,\delta)$ is a world as defined above.

Let $\omega\in\Omega$.

For $a,b\in A$, we note $a\sim_\omega b$ if $\delta(\omega,a)=1\iff\delta(\omega,b)=1$.

$\sim_\omega$ is an equivalence relation on $A$.

It's obvious.

We note $A_\omega=A/\sim_\omega$ and $\delta_\omega:A_\omega\to\{0,1\}$, $[a]\mapsto \delta(\omega,a)$.

For $a,b\in A$, we say $a$ senses $b$ if $\exists\omega\in\Omega$, $\delta_\omega(a)=1\implies\delta_\omega(b)=1$. We note $a\mathcal{S}b$.

$\mathcal{S}_\omega$ is a total order on $A_\omega$.

It's obvious.
